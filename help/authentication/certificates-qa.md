# 證書問答 {#certificates-q}

>[!NOTE]
>
>此頁面的內容僅供參考。 若要使用此API，必須具備目前的Adobe授權。 不允許未經授權使用。

</br>

**問題1:** 可以在iOS和Android之間註冊憑證嗎？

**答：** iOS和Android的憑證與目前設定相同。 原生憑證用於兩個平台。

</br>

**問題2:** 可以在生產和測試環境中將相同的iOS憑證用作主要和備份憑證嗎？ 如果不建議使用，可以說明嗎？

**答：** 配置與主證書和備份證書相同的證書沒有任何意義。 我們有主證書和備份證書的概念，這樣我們就可以為程式設計師配置多個證書，以防主證書過期或被撤銷。 擁有備份證書將給程式設計師提供更改主證書的時間，而不會影響發行環境。 但您可以對「生產」和「預備」設定檔使用相同的一組主要和備份憑證。

</br>

**第3季：** 使用新靈活TempPass的網頁是否需要新憑證？ 

**答：** 證書（以及任何證書）是在媒體公司和程式設計師級別配置的。 FlexibleTempPass是MVPD，您不需要為它配置任何證書，因此，如果您將現有程式設計師與靈活的TempPass整合，則將使用已在程式設計師/媒體公司級別配置的證書。
