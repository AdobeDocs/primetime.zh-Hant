# 客戶端整合{#client-integration}

為了引導客戶機針對本地個性化伺服器(而不是Adobe托管的全局個性化伺服器)進行個性化，客戶機應利用先前建立的本地DRM元資料。 使非個性化客戶端使用特殊元資料執行許可證獲取或初始化DRM將導致客戶端連接到定制個性化伺服器URL。

示例代碼段包含在 [!DNL client_sample] 的子菜單。
