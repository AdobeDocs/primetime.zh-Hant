# 更新租使用者設定檔{#updating-the-tenant-configuration-file}

中指定的所有值 [!DNL flashaccess-tenant.xml] 可隨時修改，而變更會在下次伺服器重新載入組態檔時生效。 此外，伺服器也會檢查所有認證(.pfx)檔案中的變更，以及Packager允許清單憑證檔案（在租使用者設定檔案中參照）。
