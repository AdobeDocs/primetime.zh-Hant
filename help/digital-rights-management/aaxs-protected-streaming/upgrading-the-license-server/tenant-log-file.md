# 租用戶日誌檔案{#tenant-log-file}

每個租用戶的租用戶日誌檔案[!DNL flashaccess-tenant.log]位於&#x200B;*LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*tenantname*&#x200B;中。 租用戶記錄檔包含稽核資訊，說明為此租用戶產生的每個授權。
