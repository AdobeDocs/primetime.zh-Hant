# 租使用者記錄檔{#tenant-log-file}

每個租使用者的租使用者記錄檔， [!DNL flashaccess-tenant.log]，位於 *LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*租使用者名稱稱*. 租使用者記錄檔包含稽核資訊，說明為此租使用者產生的每個授權。
