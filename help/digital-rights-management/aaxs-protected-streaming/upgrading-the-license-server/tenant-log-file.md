# 租戶日誌檔案{#tenant-log-file}

每個租戶的租戶日誌檔案， [!DNL flashaccess-tenant.log]，位於 *LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*tenantname*。 租戶日誌包含描述為此租戶生成的每個許可證的審核資訊。
