# 日誌檔案{#log-files}

Adobe Access Server for Protected Streaming應用程式產生的記錄檔將位於&#x200B;*LicenseServer.LogRoot*&#x200B;所指定的目錄中。

>[!NOTE]
>
>如果當前日誌檔案在伺服器運行時被刪除或移動，則可能不會重新建立日誌檔案，並且某些日誌資訊將丟失。
