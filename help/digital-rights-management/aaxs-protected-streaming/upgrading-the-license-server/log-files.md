# 記錄檔{#log-files}

Adobe Access Server為受保護的串流應用程式產生的記錄檔將位於指定的目錄中 *LicenseServer.LogRoot*.

>[!NOTE]
>
>如果在伺服器執行期間刪除或移動目前的記錄檔，則可能無法重新建立記錄檔，並且某些記錄資訊將會遺失。
