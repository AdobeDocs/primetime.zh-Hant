# 日誌檔案{#log-files}

由Adobe Access Server為受保護的流應用程式生成的日誌檔案將位於&#x200B;*LicenseServer.LogRoot*&#x200B;指定的目錄中。

>[!NOTE]
>
>如果當前日誌檔案在伺服器運行時被刪除或移動，則可能不會重新建立日誌檔案，並且某些日誌資訊將丟失。
