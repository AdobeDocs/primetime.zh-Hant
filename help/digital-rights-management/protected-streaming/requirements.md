# 要求{#requirements}

您的構建環境必須滿足以下最低要求：

* MicrosoftWindows Server 2008或Red Hat® Enterprise Linux® 5.6
* OracleJava JRE 1.6(建立自定義授權擴展需要OracleJava JDK 1.6)
* Apache Tomcat® 6，在 [!DNL Third Party\Tomcat\6.0.18] DVD的資料夾
* Adobe頒發的憑據
