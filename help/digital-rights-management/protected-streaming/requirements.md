# 需求{#requirements}

您的組建環境必須符合下列最低需求：

* Microsoft Windows Server 2008或Red Hat® Enterprise Linux® 5.6
* oracleJava JRE 1.6 (建立自訂授權擴充功能需要OracleJava JDK 1.6)
* Apache Tomcat® 6可在 [!DNL Third Party\Tomcat\6.0.18] DVD的資料夾
* Adobe已核發的認證
