# 需求{#requirements}

您的構建環境必須符合以下最低要求：

* Microsoft Windows Server 2008或Red Hat® Enterprise Linux® 5.6
* Oracle Java JRE 1.6（建立自定義授權擴展時需要Oracle Java JDK 1.6）
* DVD資料夾中提供的Apache Tomcat® 6 [!DNL Third Party\Tomcat\6.0.18] 。
* Adobe已核發的認證
