# 要求{#requirements}

您的構建環境必須符合以下最低要求：

* Microsoft Windows Server 2008或Red Hat® Enterprise Linux® 5.6
* OracleJava JRE 1.6(OracleJava JDK 1.6是建立自訂授權擴充功能的必要條件)
* DVD的[!DNL Third Party\Tomcat\6.0.18]資料夾中提供的Apache Tomcat® 6
* Adobe頒發的證書
