# 概觀 {#overview}

Adobe Primetime DRM參考實作命令列工具可讓您直接從命令列執行各種DRM相關工作。

Primetime DRM參照實施包括下列命令列工具：

* 原則管理員 — 建立和管理DRM原則
* Media Packager — 建立加密檔案
* 原則更新清單管理員 — 建立並顯示DRM原則更新清單
* 撤銷清單管理員 — 建立與顯示撤銷清單
* AIR發行者ID — 計算AIR應用程式的發行者ID
* 公用程式授權產生器 — 在本機產生授權（不需要向伺服器提出授權要求）
* 授權內嵌程式 — 將預先產生的授權內嵌至內容中
