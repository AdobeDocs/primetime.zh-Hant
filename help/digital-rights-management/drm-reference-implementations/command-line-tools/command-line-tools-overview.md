# 概述 {#overview}

使用Adobe PrimetimeDRM參考實現命令行工具，您可以直接從命令行執行與DRM相關的各種任務。

黃金時段DRM參考實現包括以下命令行工具：

* 策略管理器 — 建立和管理DRM策略
* 介質打包器 — 建立加密檔案
* 策略更新清單管理器 — 建立和顯示DRM策略更新清單
* 吊銷清單管理器 — 建立和顯示吊銷清單
* AIR發佈者ID — 計算AIR應用程式的發佈者ID
* Utility License Generator — 本地生成許可證（不向伺服器發出許可證請求）
* 許可證嵌入器 — 將預生成的許可證嵌入內容
