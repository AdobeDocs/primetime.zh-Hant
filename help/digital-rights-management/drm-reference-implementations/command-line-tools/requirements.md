# 命令行工具要求{#command-line-tools-requirements}

* Java 1.5或更新版本。
* 由Adobe核發的封裝、傳輸和授權伺服器憑證（憑證和密碼）。

   這些憑證可用來加密和簽署視訊檔案、簽署「原則更新」和「撤銷」清單，以及預先產生授權。

>[!NOTE]
>
>由於Java錯誤，您在命令行上鍵入的任何引數（如檔案名、DRM策略名或說明）只能使用作業系統預設字元集中的字元。
