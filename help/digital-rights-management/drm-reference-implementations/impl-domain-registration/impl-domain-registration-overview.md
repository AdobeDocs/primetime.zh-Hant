# 概觀 {#implement-domain-registration-overview}

參考實作授權伺服器示範網域註冊商業邏輯上的兩種變化：

* **以身分為基礎**  — 每個已驗證身分的使用者都有網域和可加入網域的特定裝置數量。
* **匿名**  — 大量裝置屬於一個網域，可能不需要驗證。
