# 概述{#implement-domain-registration-overview}

參考實施授權伺服器展示網域註冊商業邏輯的兩種變化：

* **基於身份** -每個已驗證用戶都有一個域和可加入域的特定設備數。
* **匿名** -許多裝置屬於一個網域，可能不需要驗證。
