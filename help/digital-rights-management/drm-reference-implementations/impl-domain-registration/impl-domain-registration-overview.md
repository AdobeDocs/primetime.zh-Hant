# 概述 {#implement-domain-registration-overview}

參考實現許可證伺服器演示了兩種域註冊業務邏輯變化：

* **基於身份**  — 每個經過身份驗證的用戶都有一個域和可加入該域的特定數量的設備。
* **匿名**  — 大量設備屬於一個域，可能不需要身份驗證。
