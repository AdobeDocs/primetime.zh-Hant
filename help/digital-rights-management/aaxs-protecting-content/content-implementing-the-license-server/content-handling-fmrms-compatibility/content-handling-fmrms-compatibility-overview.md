# 概述 {#handling-fmrms-compatibility-overview}

與Flash媒體Rights Management伺服器1.x相容性相關的請求有兩種類型。 一種請求類型用於提示1.x客戶端升級到支援AdobeAccess 2.0或更高版本的運行時。 另一個用於在請求許可證之前將1.x元資料更新為Adobe訪問格式。 只有在以前使用FMRMS 1.0或1.5部署內容時，才需要支援這些請求。
