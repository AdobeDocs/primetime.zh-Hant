# 使用者驗證 {#user-authentication}

指定取得授權是否需要認證，例如使用者名稱和密碼。 如果指定了驗證（以身分為基礎）授權，則伺服器會在發行授權之前驗證使用者。

**範例使用案例：** 訂閱服務在發行內容授權之前可能需要輸入使用者名稱/密碼。 DVD或Blu-ray Disc搭配Digital Copy可能會提供程式碼或其他代號作為付款證明，可兌換成電子下載。
