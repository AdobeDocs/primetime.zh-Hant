# 用戶驗證 {#user-authentication}

指定是否需要憑據（如用戶名和密碼）來獲取許可證。 如果指定了經過驗證（基於身份）的許可，則伺服器在頒發許可證之前驗證用戶。

**示例用例：** 訂閱服務可能要求在頒發內容許可證之前輸入用戶名/密碼。 帶有Digital Copy的DVD或藍光光碟可能提供代碼或其他令牌作為付款憑證，可以兌換為電子下載。
