# 自定義資料 {#custom-data}

下表介紹了「自定義資料」首選項：

| 首選項 | 說明 |
|---|---|
| 自定義策略屬性 | 指定許可證伺服器在頒發許可證時可能使用的自定義屬性。 |
| 自定義許可證屬性 | 指定自定義屬性，這些屬性將出現在發給客戶端的許可證中。 客戶端應用程式將有權訪問這些屬性。 |
