# 自訂資料 {#custom-data}

下表說明「自訂資料」偏好設定：

| 偏好設定 | 說明 |
|---|---|
| 自訂原則屬性 | 指定授權伺服器在發行授權時可能會使用的自訂屬性。 |
| 自訂授權屬性 | 指定自訂屬性，這些屬性會顯示在發給用戶端的授權中。 客戶端應用程式可以訪問這些屬性。 |
