# 自訂資料 {#custom-data}

下表說明「自訂資料」偏好設定：

| 偏好設定 | 說明 |
|---|---|
| 自訂原則屬性 | 指定授權伺服器發行授權時可使用的自訂屬性。 |
| 自訂授權屬性 | 指定自訂屬性，這些屬性將顯示在頒發給使用者端的授權中。 使用者端應用程式將可以存取這些屬性。 |
