# 建立新策略{#create-a-new-policy}

必須先建立一或多個原則，才能封裝任何內容。 如需原則中可能指定之使用規則的概述，請參閱Adobe Access SDK概述中的使用規則。

要建立新策略，請按一下「新建」並輸入策略名稱。 填寫所需的原則屬性（所有設定都是選用的）。 完成後，按一下&#x200B;**[!UICONTROL Save]**。 策略將保存為資源目錄中的[!DNL policyname.pol]。
