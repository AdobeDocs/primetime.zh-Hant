# 更新現有策略 {#update-an-existing-policy}

要更新現有策略，請從下拉清單中選擇檔案名，然後按一下 **[!UICONTROL Open]**。 修改所需的策略屬性。 除與「驗證」和「許可證連結」相關的屬性外，所有屬性都可以修改。

完成後，按一下 **[!UICONTROL Save]**。 中的策略檔案 [!DNL Resource] 目錄將替換為更新的版本。

>[!NOTE]
>
>即使策略名稱已更改， [!DNL Resource] 將不修改目錄。
