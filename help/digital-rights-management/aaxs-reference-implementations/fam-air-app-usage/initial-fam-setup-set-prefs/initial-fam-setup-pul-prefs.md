# 原則更新清單喜好設定 {#policy-update-list-preferences}

此標籤包含建立原則更新清單所需的設定。 下表說明偏好設定：

| 偏好設定 | 說明 |
|---|---|
| 授權伺服器認證 | 授權伺服器認證，由Adobe簽發。 此認證用於簽署原則更新清單。 |
| 檔案名稱 | PKCS#12 ( [!DNL .pfx])檔案，包含憑證和私密金鑰。 檔案必須位於 [!DNL Resource] 目錄。 |
| 檔案密碼 | 的密碼 [!DNL .pfx] 檔案 |
