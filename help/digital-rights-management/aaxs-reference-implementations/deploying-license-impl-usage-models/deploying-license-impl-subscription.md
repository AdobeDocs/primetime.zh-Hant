# 訂閱{#subscription}

有些服務提供付費訂閱，讓使用者只要繼續支付月費，就能無限制地存取大型內容庫。 授權伺服器會針對每項內容發出唯一的授權，並會發出根授權，其到期時間與訂閱期間一致。 每個月，當使用者重新訂閱時，根授權也可以續約。 當要求授權時，使用者需要驗證，如此伺服器就可確認使用者的訂閱是最新版本。
