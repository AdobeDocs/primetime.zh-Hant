# 下載至擁有 {#download-to-own}

使用下載至擁有(DTO)使用模式時，使用者可以下載內容以供線上或離線使用，並獲得內容的永久授權。 請求授權時，使用者必須驗證，讓伺服器可以驗證使用者是否已購買內容。
