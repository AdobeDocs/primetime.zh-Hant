# 下載至擁有{#download-to-own}

使用「下載至擁有」(DTO)使用模式時，使用者可下載內容以供線上或離線使用，並獲得內容的永久授權。 當要求授權時，使用者必須進行驗證，如此伺服器才能確認使用者已購買內容。
