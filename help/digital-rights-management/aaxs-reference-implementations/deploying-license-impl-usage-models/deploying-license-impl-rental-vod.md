# 租賃/隨選視訊 {#rental-video-on-demand}

透過隨選影片(VOD)使用模式，提供內容時會有時間限制。 例如，使用者可以選擇在30天內播放內容；但是，一旦播放開始，使用者最多有48小時完成觀看，之後內容將無法播放。 請求授權時，使用者必須驗證，讓伺服器可以驗證使用者是否有租用的帳戶。
