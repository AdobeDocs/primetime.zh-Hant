# 租賃／隨選視訊 {#rental-video-on-demand}

透過隨選視訊(VOD)使用模型，內容會受到時間限制。 例如，使用者可以選擇在30天內播放內容；不過，當播放開始時，使用者最多有48小時可以完成觀看，之後內容將無法再播放。 在請求授權時，使用者必須進行驗證，如此伺服器才能確認使用者有租用帳戶。
