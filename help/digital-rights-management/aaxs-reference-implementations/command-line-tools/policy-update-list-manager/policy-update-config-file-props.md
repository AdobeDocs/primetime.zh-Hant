# 配置檔案屬性 {#configuration-file-properties}

以下是「策略更新清單管理器」屬性，它指定包含簽名吊銷清單（許可證伺服器證書）憑據的PKCS12檔案：

* `revocation.sign.certfile=license-server-credentials.pfx`
* `revocation.sign.certpass=password`
