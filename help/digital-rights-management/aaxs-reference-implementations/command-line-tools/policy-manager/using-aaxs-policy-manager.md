# 使用策略管理器{#using-policy-manager}

使用策略管理器，您可以建立和管理策略。 運行策略管理器之前，可以選擇在配置檔案中設定策略管理器屬性的值。 配置檔案指定將應用於所有策略的資訊。 也可以在命令行上指定所有策略管理器屬性。
