# 使用策略管理器 {#using-policy-manager}

使用策略管理器，可以建立和管理策略。 在運行策略管理器之前，可選地在配置檔案中為策略管理器屬性設定值。 配置檔案指定將應用於所有策略的資訊。 也可以在命令行上指定所有策略管理器屬性。
