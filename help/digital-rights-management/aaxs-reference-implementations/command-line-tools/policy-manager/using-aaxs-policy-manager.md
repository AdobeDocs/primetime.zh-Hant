# 使用原則管理員 {#using-policy-manager}

您可以使用Policy Manager來建立和管理原則。 執行Policy Manager之前，您可以選擇在組態檔中設定Policy Manager屬性的值。 組態檔會指定將套用至所有原則的資訊。 也可以在命令列上指定所有Policy Manager屬性。
