# 授權產生器概觀 {#license-generator-overview}

使用License Generator命令列工具，您就可產生授權，而不需要用戶端傳送授權要求至伺服器。 預先產生的授權可內嵌在內容中，或透過其他機制（例如簡單的HTTP網路伺服器）傳送至用戶端。
