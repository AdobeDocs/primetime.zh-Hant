# 授權產生器概述 {#license-generator-overview}

使用「許可證產生器」命令列工具，您可以產生許可證，而不要求使用者端將許可證請求傳送到伺服器。 預先產生的授權可內嵌於內容中，或透過其他機制（例如簡單的HTTP網頁伺服器）傳送給使用者端。
