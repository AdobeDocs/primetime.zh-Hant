# 要求 {#requirements}

您需要確保安裝了以下元件：

* Tomcat 6.0或更高版本
* 資料庫（如MySQL）（僅許可證伺服器元件需要）
* Java 1.6或更高版本
* Ant 1.8或更高版本（使用示例生成指令碼）

安裝Tomcat和MySQL後，請從Adobe獲取憑據。
