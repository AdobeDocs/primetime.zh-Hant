# 需求 {#requirements}

您必須確保已安裝下列專案：

* Tomcat 6.0或更新版本
* MySQL之類的資料庫（僅適用於License Server元件）
* Java 1.6或更新版本
* Ant 1.8或更新版本（使用範例建置指令碼）

安裝Tomcat和MySQL後，請從Adobe取得認證。
