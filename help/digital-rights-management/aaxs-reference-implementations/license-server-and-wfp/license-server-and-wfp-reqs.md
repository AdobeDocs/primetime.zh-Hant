# 要求{#requirements}

您需要確保已安裝下列產品：

* Tomcat 6.0或更新版本
* MySQL之類的資料庫（僅許可證伺服器元件需要）
* Java 1.6或更新版本
* Ant 1.8或更新版本（以使用範例建置指令碼）

安裝Tomcat和MySQL後，請從Adobe獲取憑據。
