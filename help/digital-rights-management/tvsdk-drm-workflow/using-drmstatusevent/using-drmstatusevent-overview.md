# 概述{#using-the-drmstatusevent-class-overview}

`DRMStatusEvent`物件會在Primetime DRM保護的內容開始成功播放時傳送。 （成功表示已驗證授權，且使用者已通過驗證並授權檢視內容）。

`DRMStatusEvent`物件包含與授權相關的資訊，包括授權是否可離線使用，或是授權過期且內容無法再檢視。
