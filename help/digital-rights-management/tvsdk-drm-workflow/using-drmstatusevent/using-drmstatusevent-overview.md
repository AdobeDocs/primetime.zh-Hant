# 概觀 {#using-the-drmstatusevent-class-overview}

A `DRMStatusEvent` 當Primetime DRM保護的內容開始成功播放時，會傳送物件。 （成功代表授權已驗證，且使用者已驗證並獲授權檢視內容）。

此 `DRMStatusEvent` 物件包含與授權相關的資訊，包括授權是否可離線使用，或授權到期且無法再檢視內容時。
