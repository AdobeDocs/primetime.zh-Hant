# 使用DRMStatusEvent類概述 {#using-the-drmstatusevent-class-overview}

當受 `DRMStatusEvent` Primetime DRM保護的內容開始成功播放時，會傳送物件。 （成功表示已驗證授權，且使用者已通過驗證並授權檢視內容）。

物 `DRMStatusEvent` 件包含與授權相關的資訊，包括授權是離線提供，還是授權過期，而內容無法再檢視。
