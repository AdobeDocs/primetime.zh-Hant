# 設定SSL {#configuring-ssl}

只有在您需要支援Flash MediaRights Management伺服器1.x使用者端時，才需要安全通訊端層(SSL)。 Adobe存取金鑰伺服器需要具有使用者端驗證的SSL。 如需詳細資訊，請參閱*使用Adobe存取金鑰伺服器*。
