# 配置SSL {#configuring-ssl}

只有在需要支援Flash媒體Rights Management伺服器1.x客戶端時，才需要安全套接字層(SSL)。 Adobe訪問密鑰伺服器需要使用客戶端身份驗證的SSL。 請參閱*使用Adobe訪問密鑰伺服器*瞭解詳細資訊。
