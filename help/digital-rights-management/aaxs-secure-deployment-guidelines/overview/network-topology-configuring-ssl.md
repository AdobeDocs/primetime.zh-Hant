# 設定SSL {#configuring-ssl}

只有在您需要支援Flash Media Rights Management Server 1.x用戶端時，才需要安全通訊端層(SSL)。 Adobe Access Key Server需要具備用戶端驗證的SSL。 如需詳細資訊，請參閱*使用Adobe Access Key Server *。
