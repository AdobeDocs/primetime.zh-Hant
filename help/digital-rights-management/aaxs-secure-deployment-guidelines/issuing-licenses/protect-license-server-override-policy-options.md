# 覆蓋策略選項{#overriding-policy-options}

發放許可證時，許可證伺服器可以覆蓋策略中指定的使用規則。 如果原則指定開始日期，則不會在該開始日期之前產生授權。 不過，在產生授權後，仍可在授權中設定未來的開始日期。 使用此選項時應小心謹慎，因為用戶端不會阻止使用者將系統時間向前移動以規避開始日期。
