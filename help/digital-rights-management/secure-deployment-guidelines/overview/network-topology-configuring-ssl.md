# 配置SSL {#configuring-ssl}

只有在需要支援Flash媒體Rights Management伺服器1.x客戶端時，才需要安全套接字層(SSL)。

Adobe PrimetimeDRM密鑰伺服器需要使用客戶端身份驗證的SSL。 有關詳細資訊，請參見 [使用Adobe PrimetimeDRM密鑰伺服器](../../using-the-drm-key-server/requirements.md)。
