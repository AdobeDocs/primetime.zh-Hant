# 加密指令碼資料 {#encrypting-script-data}

此封裝選項會指定是否加密內嵌在內容中的指令碼資料。

>[!NOTE]
>
>此規則僅適用於FLV檔案格式。 對於其他格式的檔案，指令碼資料一律保留在清除中。

使用案例範例：使用此選項可讓指令碼資料保持未加密，讓中繼資料彙總工具可讀取受保護內容的中繼資料。
