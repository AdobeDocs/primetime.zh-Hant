# 加密指令碼資料 {#encrypting-script-data}

此打包選項指定是否加密嵌入內容中的指令碼資料。

>[!NOTE]
>
>此規則僅適用於FLV檔案格式。 對於其他格式的檔案，指令碼資料始終保留為清除。

示例用例：使用此選項可使指令碼資料保持未加密狀態，這允許元資料聚合工具讀取受保護內容的元資料。
