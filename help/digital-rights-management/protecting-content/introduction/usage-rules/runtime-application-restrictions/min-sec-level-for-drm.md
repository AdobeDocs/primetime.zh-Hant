# DRM和執行階段的最低安全性等級{#minimum-security-level-for-drm-and-runtimes}

最低安全性等級指定存取內容所需的安全性等級。 您可以為每個元件指定個別的安全性層級。

DRM/Runtime模組的預設安全性等級為10000。

使用案例範例：某些型別的內容（例如HD視訊）可能需要比其他型別更高的安全性等級。
