# DRM和運行時的最低安全級別{#minimum-security-level-for-drm-and-runtimes}

最低安全級別指定訪問內容所需的安全級別。 為每個元件指定各個安全級別。

DRM/運行時模組的預設安全級別為10000。

示例用例：某些類型的內容（例如高清視頻）可能需要比其他類型更高的安全級別。
