# 相對結束日期{#relative-end-date}

相對結束日期會指定授權到期日，此日期會相對於封裝日期而顯示，而非相對於授權簽發日期。

使用案例範例：在自動封裝程式中，針對一系列影片使用具有此選項的單一Primetime DRM原則，將到期日設定為相對於封裝日期的30天。
