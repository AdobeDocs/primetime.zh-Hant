# 相對結束日期{#relative-end-date}

相對結束日期指定許可證到期日期，該日期與打包日期相比表示，而不是與頒發許可證的日期相比。

示例用例：在自動打包過程中，對一系列視頻使用單個黃金時段DRM策略和此選項，將到期日設定為相對於打包日期的30天。
