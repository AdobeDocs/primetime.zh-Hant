# 概述 {#configure-the-player-user-interface-overview}

使用TVSDK，您可以控制即時視頻點播(VOD)的基本播放體驗。 TVSDK不會為您配置播放器，而是在播放器實例上提供可用於配置播放器用戶介面的方法和屬性。
