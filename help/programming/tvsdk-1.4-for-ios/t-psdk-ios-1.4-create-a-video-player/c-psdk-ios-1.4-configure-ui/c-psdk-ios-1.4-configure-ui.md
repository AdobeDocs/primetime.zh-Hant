# 概述{#configure-the-player-user-interface-overview}

透過TVSDK，您可以控制即時和隨選視訊(VOD)的基本播放體驗。 TVSDK不會為您設定播放器，而是提供播放器例項上的方法和屬性，供您用來設定播放器使用者介面。
