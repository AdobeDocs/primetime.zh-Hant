# 使用預設播放行為 {#use-the-default-playback-behavior}

您可以選擇使用預設的廣告行為。

使用預設行為：

    *如果您實作自己的「AdvertisingFactory」類別，請為「createAdPolicySelector」傳回null 。
    
    *如果您沒有「AdvertisingFactory」類別的自訂實作，TVSDK會使用預設廣告原則選擇器。
