# 播放事件{#playback-events}

TVSDK會在媒體播放作業發生時（例如視訊開始播放）傳送播放事件。

若要收到有關所有播放相關事件的通知，請註冊以下實作： `MediaPlayer.PlaybackEventListener`，包括下列事件回呼。
