# 媒體播放和容錯移轉 {#media-playback-and-failover}

對於即時和隨選影片(VOD)媒體，TVSDK會透過下載與中階解析度位元速率相關的播放清單來開始播放，並下載該播放清單定義的媒體區段。 它會快速選取高解析度位元速率播放清單及其相關媒體，並繼續下載程式。
