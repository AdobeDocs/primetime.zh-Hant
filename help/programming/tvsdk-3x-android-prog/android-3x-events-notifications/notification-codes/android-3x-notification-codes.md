# 通知代碼 {#notification-codes}

TVSDK通知系統會產生各種錯誤、警告和資訊通知，以提供診斷中繼資料。

通知物件提供與播放器狀態相關的資訊。 TVSDK提供按時間順序排序的通知物件清單。 每個通知都包含下列中繼資料：

  </tr>
