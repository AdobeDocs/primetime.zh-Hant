# 調試和記錄{#debugging-and-logging}

瀏覽器TVSDK預設啟用日誌記錄。 它還提供API支援，以啟用或禁用SDK中的日誌記錄。 使用AdobePSDK.LogManager.enableLogging API啟用或禁用日誌。
