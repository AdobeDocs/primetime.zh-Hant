# 偵錯和記錄{#debugging-and-logging}

瀏覽器TVSDK預設為啟用記錄功能。 此外，也提供API支援，可啟用或停用來自SDK的記錄功能。 使用AdobePSDK.LogManager.enableLogging API來啟用或停用記錄檔。
