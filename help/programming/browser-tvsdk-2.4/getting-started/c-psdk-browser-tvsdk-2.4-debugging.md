# 除錯和記錄{#debugging-and-logging}

瀏覽器TVSDK依預設啟用記錄。 此外，它還提供API支援，以啟用或停用SDK記錄。 使用AdobePSDK.LogManager.enableLogging API啟用或停用記錄檔。
