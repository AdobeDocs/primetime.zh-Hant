# 設定播放器{#skinning-the-player}的外觀

您可使用下列資訊來協助您設定播放器外觀。 對於每個視覺結構，預設行為中都會提到相應的行為。

>[!IMPORTANT]
>
>本檔案中的外觀設定詳細資料是針對由UI架構建立的預設UI元素。 如果您的播放器修改了這些元素，您也必須變更外觀元素。
