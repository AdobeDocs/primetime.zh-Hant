# 概述{#requirements-overview}

瀏覽器TVSDK需要媒體內容、資訊清單內容和軟體版本的特定屬性。

瀏覽器TVSDK發行包含下列播放器：

* `Primetime.js` 用於HLS案頭版和移動版

播放器包含下列參考API:

* `Primetime.js` 瀏覽器TVSDK
* 核心播放的JavaScript API
* 參考示範播放程式
