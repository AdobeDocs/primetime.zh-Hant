## DASH進階廣告插入功能(CSAI) {#dash-advanced-insertion-features}

| Ad Insertion | VOD | 僅限廣告 | 不支援 |
| Ad Insertion | VOD | 目標定位引數 | 僅限VOD |
| Ad Insertion | VOD | 自訂引數 | 僅限VOD |

| Ad Insertion | VOD +即時 | 延遲廣告載入 | 不支援 |

| Ad Insertion | VOD | VPAID 2.0 | 不支援 |
