# 使用即時主資訊清單更新{#use-live-master-manifest-update}

您可以開啟此功能並檢查相關事件。

1. 若要開啟即時主資訊清單更新，請設定屬性以設定更新頻率(以分鐘為 `NetworkConfiguration.masterUpdateInterval` 單位)。
1. （可選）監聽事件以追蹤成功的資訊清單 `MediaPlayerItemEvent.MASTER_UPDATED` 更新。
