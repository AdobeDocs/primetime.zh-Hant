# 使用即時主要資訊清單更新{#use-live-master-manifest-update}

您可以開啟此功能並檢查相關事件。

1. 若要開啟即時主要資訊清單更新，請透過設定 `NetworkConfiguration.masterUpdateInterval` 屬性。
1. 或者，您可以偵聽 `MediaPlayerItemEvent.MASTER_UPDATED` 事件。
