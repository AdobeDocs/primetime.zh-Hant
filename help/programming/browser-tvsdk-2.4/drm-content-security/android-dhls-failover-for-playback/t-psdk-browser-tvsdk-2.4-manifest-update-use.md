# 使用即時主清單更新{#use-live-master-manifest-update}

您可以開啟此功能並檢查相關事件。

1. 要啟用即時主清單更新，請通過設定 `NetworkConfiguration.masterUpdateInterval` 屬性。
1. （可選）通過偵聽 `MediaPlayerItemEvent.MASTER_UPDATED` 的子菜單。
