# 隱藏式字幕樣式選項{#closed-caption-styling-options}

您可以指定數個註解樣式選項，這些選項會覆寫原始註解中的樣式選項。

```js
new TextFormat( 
   font,  
   fontColor,  
   edgeColor,  
   fontEdge,  
   backgroundColor,  
   fillColor,  
   size,  
   fontOpacity,  
   backgroundOpacity,  
   fillOpacity, 
   bottomInset, 
   safeArea) 
```

>[!TIP]
>
>在定義預設值的選項中(例如， `DEFAULT`)，該值是指最初指定註解時的設定。
