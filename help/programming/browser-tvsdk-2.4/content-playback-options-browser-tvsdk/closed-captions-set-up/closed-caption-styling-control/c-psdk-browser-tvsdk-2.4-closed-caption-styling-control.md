# 概觀 {#control-closed-caption-styling-overview}

您可以使用TextFormat類別為隱藏式字幕軌跡提供樣式資訊。 這會設定播放器所顯示之任何隱藏式字幕的樣式。

此類別會封裝隱藏式字幕樣式資訊，例如字型型別、大小、顏色和背景不透明度。
