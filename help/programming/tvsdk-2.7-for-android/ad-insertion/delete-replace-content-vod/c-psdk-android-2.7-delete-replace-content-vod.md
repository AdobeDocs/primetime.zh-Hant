# 刪除和替換VOD流中的廣告 {#delete-and-replace-ads-in-vod-streams}

TVSDK支援VOD流中廣告內容的寫程式刪除和替換。

刪除和替換特徵擴展了自定義和標籤特徵。 自定義廣告標籤將主內容的節標籤為與廣告相關的內容句點。 除了標籤這些時間範圍外，還可以刪除和替換時間範圍。
