# 概觀 {#clickable-ads-overview}

TVSDK會提供您資訊，讓您在點進廣告上採取行動。 建立播放器UI時，您必須決定當使用者點按可點按廣告時如何回應。

若是Android適用的TVSDK，只能點按線性廣告。
