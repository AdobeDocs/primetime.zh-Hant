# 概述{#content-protection-overview}

您可以使用Primetime數位版權管理(DRM)系統的功能，提供對視訊內容的安全存取。 或者，您也可以使用協力廠商的DRM解決方案，做為Adobe整合解決方案的替代方案。

如需協力廠商DRM解決方案可用性的最新資訊，請連絡您的Adobe代表。
