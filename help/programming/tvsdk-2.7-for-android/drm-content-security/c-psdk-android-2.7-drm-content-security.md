# 概述{#content-protection-overview}

您可以使用PrimetimeDigital Rights Management(DRM)系統的功能，提供對視訊內容的安全存取。 或者，您也可以使用協力廠商DRM解決方案來取代Adobe的整合解決方案。

請洽詢您的Adobe代表，以取得有關協力廠商DRM解決方案可用性的最新資訊。
