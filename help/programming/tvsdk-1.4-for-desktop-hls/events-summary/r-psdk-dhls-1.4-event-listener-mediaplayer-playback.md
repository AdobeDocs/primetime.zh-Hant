# 播放事件 {#playback-events}

您的應用程式可監聽TVSDK傳送的事件，以監控播放器中的活動和播放器的變更狀態。

TVSDK會在媒體播放作業發生時（例如視訊開始播放）傳送播放事件。 若要收到有關所有播放相關事件的通知，請使用 `MediaPlayer` 物件。
