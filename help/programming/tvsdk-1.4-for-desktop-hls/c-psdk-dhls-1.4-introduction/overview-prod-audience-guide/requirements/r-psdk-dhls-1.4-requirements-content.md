# 內容和清單要求 {#content-and-manifest-requirements}

檢查流和播放清單（清單）（包括DRM加密密鑰）的限制和要求。

| 包括並設定 `RESOLUTION` 清單檔案中每個ABR流的屬性。 必須使用Flash Player14或更高版本。 |
|---|
| 如果受DRM保護的流是多比特率(MBR)，則用於MBR的DRM加密密鑰應與所有比特率流中使用的密鑰相同。 |
| 必須與主內容的格式副本具有相同的比特率格式副本。 |
