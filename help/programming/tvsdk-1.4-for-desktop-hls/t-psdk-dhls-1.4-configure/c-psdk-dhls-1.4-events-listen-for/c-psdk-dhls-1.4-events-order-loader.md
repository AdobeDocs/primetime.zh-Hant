# 載入器事件{#loader-events}

您的播放器可以根據下列事件實作動作：

| 事件 | 意義 |
|---|---|
| `MediaPlayerItemLoader.completed` | 媒體資源載入成功完成。 |
| `MediaPlayerItemLoader.failed` | 載入介質資源時發生問題。 |
