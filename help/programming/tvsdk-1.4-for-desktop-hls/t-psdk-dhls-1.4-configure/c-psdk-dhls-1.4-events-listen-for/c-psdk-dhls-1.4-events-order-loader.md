# 載入程式事件{#loader-events}

您的播放器可以基於以下事件實施操作：

| 事件 | 意義 |
|---|---|
| `MediaPlayerItemLoader.completed` | 媒體資源載入已成功完成。 |
| `MediaPlayerItemLoader.failed` | 載入媒體資源時出現問題。 |
