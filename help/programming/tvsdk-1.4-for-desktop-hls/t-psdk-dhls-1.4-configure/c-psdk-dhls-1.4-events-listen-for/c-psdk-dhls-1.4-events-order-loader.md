# 載入器事件{#loader-events}

您的播放器可根據下列事件實作動作：

| 事件 | 含義 |
|---|---|
| `MediaPlayerItemLoader.completed` | 媒體資源載入已成功完成。 |
| `MediaPlayerItemLoader.failed` | 媒體資源載入時發生問題。 |
