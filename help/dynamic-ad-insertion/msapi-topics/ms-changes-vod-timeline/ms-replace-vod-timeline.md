# VOD的變更 {#changes-to-vod}

適合VOD內容之一播放的廣告時間軸可能不適合後續播放。 您可以取代VOD時間軸，而不需變更內容。

您可能想要取代VOD時間軸的情況包括：

* 取代當地廣告，但在C3視窗中保留國家廣告。
* 在C3視窗關閉後取代燒入廣告。
* 以持續時間更長的較新廣告動態取代舊的C3廣告。
* 插入的廣告較少或完全沒有。

您可以以原始的M3U8檔案和查詢參數的不同值提交新的廣告插入請求來取代廣告時 `pttimeline` 間軸。
