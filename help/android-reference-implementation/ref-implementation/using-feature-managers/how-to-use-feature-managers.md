# 如何使用功能管理員 {#how-to-use-feature-managers}

可使用功能管理員來啟用、停用及設定功能，方法有兩種：

* 您可以將設定檔案中的設定資訊傳遞至媒體播放器，藉此建立及設定功能管理員。
* 您可以使用管理員處理站來啟用和停用功能，而不需瀏覽程式碼。
