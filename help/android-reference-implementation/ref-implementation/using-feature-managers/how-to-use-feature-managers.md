# 如何使用功能管理員 {#how-to-use-feature-managers}

可使用功能管理員來啟用、停用及設定功能，方法有兩種：

* 您可以從設定檔案將設定資訊傳遞至媒體播放器，以建立及設定功能管理員。
* 您可以使用管理員工廠來啟用和停用功能，而不需瀏覽程式碼。
