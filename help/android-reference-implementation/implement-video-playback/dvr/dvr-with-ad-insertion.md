# 廣告插入{#dvr-with-ad-insertion}的DVR

廣告分段在控制列中顯示為紅色列，如下所示：

<!--<a id="fig_720DD22D2318485EAB4BEA55C30D5ECF"></a>-->

![](assets/dvr-with-ads.jpg)

可見範圍為灰色，紅點代表廣告插播。
