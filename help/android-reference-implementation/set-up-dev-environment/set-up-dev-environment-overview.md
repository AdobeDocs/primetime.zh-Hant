# 概述{#overview}

TVSDK Primetime參考Android專案的設定方式，使其與Eclipse相容，而不需進行任何重大設定或IDE變更。 若要設定Primetime參考專案，請先依照指示設定Eclipse，然後依照步驟建立專案。
