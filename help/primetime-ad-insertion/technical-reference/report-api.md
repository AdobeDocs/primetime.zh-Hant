# 報表API {#report-api}

使用者介面為客戶和啟用(Adobe)團隊管理角色。 客戶可以存取入口網站，然後建立和編輯其設定。 使用者也可以在使用者介面上查看其廣告曝光次數的報表。

API在幕後運作，以便客戶和管理員與後端基礎架構通訊。

請參閱以下主題中的報表API檔案： [Auditude的報表API](../assets/auditude-report-api.pdf).
