# 支援的格式{#supported-formats}

PrimetimeAd Insertion會截取並支援廣告插入至下列媒體串流：

## HLS {#hls}

- 混音音訊／視訊片段播放清單
- CMAF和已解除混音的音訊／視訊播放清單，包括純音訊播放清單
- WebVTT
- I-Frame串流播放清單(#EXT-X-I-FRAME-STREAM-INF)
- Thubmanil影像播放清單(#EXT-X-IMAGE-STREAM-INF)

## 虛線{#dash}

- CMAF和已解除混音的音訊／視訊播放清單，包括純音訊播放清單
- WebVTT
