# 支援的格式 {#supported-formats}

PrimetimeAd Insertion會攔截並支援將廣告插入下列媒體串流：

## HLS {#hls}

- 將音訊/視訊片段播放清單設為靜音
- CMAF和已取消連結的音訊/視訊播放清單，包括僅限音訊的播放清單
- WebVTT
- I-Frame串流播放清單(#EXT-X-I-FRAME-STREAM-INF)
- Thubmanil影像播放清單(#EXT-X-IMAGE-STREAM-INF)

## 虛線 {#dash}

- CMAF和已取消連結的音訊/視訊播放清單，包括僅限音訊的播放清單
- WebVTT
