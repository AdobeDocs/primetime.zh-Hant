# 進階遙測 {#advanced-telemetry}

客戶可以使用PrimetimeAd Insertion來擷取廣告和工作階段相關的事件和量度，以找出內容和廣告錯誤的主要原因，進而深入瞭解視訊和廣告效能。  廣告請求可在原始請求層級擷取，讓操作員能針對直接來自提供者的特定VAST或VMAP標籤進行除錯。
