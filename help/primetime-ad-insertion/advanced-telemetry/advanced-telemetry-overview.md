# 高級遙測{#advanced-telemetry}

客戶可以使用Primetime廣告插入來擷取廣告和作業相關事件與量度，以找出內容與廣告錯誤的主要原因，從而獲得視訊與廣告效能的見解。  廣告請求可在原始請求層級擷取，讓營運商能夠對直接來自提供者的特定VAST或VMAP標籤進行除錯。
