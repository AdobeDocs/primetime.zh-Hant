# 高級遙測 {#advanced-telemetry}

客戶能夠使用黃金時段Ad Insertion捕獲廣告和會話相關事件和指標，以發現內容和廣告錯誤的主要原因，從而深入瞭解視頻和廣告效能。  廣告請求可以在原始請求級別捕獲，使運營商能夠調試直接來自提供商的特定VAST或VMAP標籤。
