# 新增內容「緩衝器」 {#adding-content-bumpers}

媒體公司可能需要在其內容周圍放置「緩衝器」，以利品牌推廣、贊助及/或符合政府規定。 PrimetimeAd Insertion提供多種方式，可以在不將緩衝器烘焙至實際內容的情況下放置緩衝器。 如需詳細資訊，請聯絡您的Primetime支援代表。
