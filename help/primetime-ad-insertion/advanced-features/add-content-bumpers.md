# 添加內容「Bumpers」 {#adding-content-bumpers}

媒體公司可能需要在其內容周圍設定「阻塞器」，以用於品牌推廣、贊助和/或滿足政府要求。 黃金時段Ad Insertion提供了多種方法，可以放置保險槓，而不將其烘焙成實際內容。 如欲瞭解更多資訊，請與您的黃金時段支援代表聯繫。
